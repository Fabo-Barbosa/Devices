# Adicionando uma senha via winbox
	Menu: System -> Users
	Campos: 
		* Existe um usuário padrão com nome admin
		* Para colocar senha em admin:
			- dois clicks em admin
			- botão [Password]
			- Preencher os campos "New Password" e "Confirm Password"
			- OK

# Adicionando um novo usuário via winbox
	Menu: System -> Users
	Campos:
		* Ícone [+]
		* preencher campos:
			- Name (nome do usuário)
			- Group (atribuir um grupo ao usuário)
			- Password
			- Confirm Password
			- Botão [Apply]
			- OK
